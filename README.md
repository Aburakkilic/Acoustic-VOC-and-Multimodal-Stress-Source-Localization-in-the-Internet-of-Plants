# Acoustic-VOC-and-Multimodal-Stress-Source-Localization-in-the-Internet-of-Plants
Physics-based simulation, sensor placement, and multimodal (acoustic + VOC) source localization pipeline for plant stress detection in the "Internet of Plants." Includes a 52-scenario dataset, greedy QR agent-plant selection, and a TDOA + physics-informed VOC fusion pipeline with extensive robustness evaluations.
